# 配置文件

::: tip
请先手动运行一次游戏. 更改设置需要在游戏关闭的情况下进行, 因此请<mark>杀后台</mark>
:::
本节展示各平台`MajdataPlay`的配置文件位置, 阅读配置相关文档时默认已阅读本节.

## Windows

与`MajdataPlay.exe`同级的目录中的`settings.json`

## iOS

<mark>一般情况下, iOS 版本不需要手动更改配置文件. 进入`设置`, 全局搜索`MajdataPlay`或在左侧列表中找到`App`并点击进入, 然后在右侧找到`MajdataPlay`. 在那里你会找到你需要的几乎所有设置.</mark>

打开`文件`App, 在目录中选择`我的设备`(如`我的iPad`), 其中会有`MajdataPlay`文件夹, 进入后有一个`settings.json`文件. 更改该文件需要使用专门的`json`编辑器, 可以在`App Store`里面搜索安装`Koder`.

## Android

- apk 文件版: `/storage/emulated/0/Documents/MajdataPlay/settings.json`
- Play 商店版: `/storage/emulated/0/Android/data/net.majdata.majdataplay/files/settings.json`
